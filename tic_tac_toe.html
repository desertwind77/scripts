minimax(depth: int, is_maximizing: bool) → int[source]

An implementation of the Minimax algorithm

Arguments:
depth (int): the current depth in the decision tree. We don’t use

this here because the decision of the tree for Tic Tac Toe is not that deep. For some other games, we may need to limit the depth that the algorithm can go to yield reasonable response time

is_maximizing (bool): operating as a maximizer or a minimizer

Returns:

(int) the score of this path. -1 and 1 for losing and winning outcomes respectively
